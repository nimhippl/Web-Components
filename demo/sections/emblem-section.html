🛡️ Emblem Component

Compact game emblems for roles, classes, factions, ranks, medals, and profile badges. Use gwc-tooltip for names and descriptions.

Rarity colors
Common Rare Epic Legendary
Emblem selection flow
Click an emblem to simulate selecting a role, faction, rank, or profile medal. The selected emblem is highlighted and its details are shown in the panel.
State variants
Same shield shape, different states. This makes locked, claimable, claimed, hidden and unlocked behavior easy to compare.
Shape variants
Different shapes for roles, classes, factions, ranks and profile emblems.
// Usage in TypeScript
import { GwcEmblem, EmblemConfig } from 'gwc';

const emblem = document.querySelector('gwc-emblem') as GwcEmblem | null;

const config: EmblemConfig = {
  id: 'defender',
  name: 'Defender',
  description: 'Block 100 attacks.',
  icon: '🛡️',
  rarity: 'epic',
  state: 'unlocked'
};

emblem?.setConfig(config);

emblem?.addEventListener('gwc-emblem-click', event => {
});

// Usage in HTML
<gwc-emblem
  theme="dark"
  size="medium"
  shape="shield"
  interactive="true"
  show-state="true">
</gwc-emblem>
